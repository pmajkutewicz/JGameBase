===GUI improvements, URL handling, Bug fixes

Date: 31.03.2012

Release (version 0.63-1) of jGameBase has been released.

<br><b>New Functionality</b>
&nbsp;&nbsp;- <i>overlays have versions</i>
&nbsp;&nbsp;- <i>system views are now saved to the database</i>
&nbsp;&nbsp;- <i>loading of images is stopped, if a new item is selected</i>
&nbsp;&nbsp;- <i>scaling animations</i>
&nbsp;&nbsp;- <i>icons with overlays for url extras</i>
&nbsp;&nbsp;- <i>context menus for extra buttons</i>
&nbsp;&nbsp;- <i>side pane slider positions remembered</i>
&nbsp;&nbsp;- <i>selected item remembered</i>
&nbsp;&nbsp;- <i>table columns remembered</i>
&nbsp;&nbsp;- <i>category windows position remembered</i>
&nbsp;&nbsp;- <i>screenshots: left click external, right click internal viewer</i>
&nbsp;&nbsp;- <i>item id displayed in status line</i>
&nbsp;&nbsp;- <i>marked extra displayed in status line</i>
&nbsp;&nbsp;- <i>preference to limit the number of extra images shown</i>
&nbsp;&nbsp;- <i>music details loaded from music file</i>
&nbsp;&nbsp;- <i>extract additional media</i>
&nbsp;&nbsp;- <i>download of url extras</i>
&nbsp;&nbsp;- <i>rudimentary ADF plugin</i>
&nbsp;&nbsp;- <i>memory usage optimized / use client JVM</i>
&nbsp;&nbsp;- <i>reorganize database after import</i>
&nbsp;&nbsp;- <i>remove duplicate urls</i>
&nbsp;&nbsp;- <i>remove orphaned extras from database</i>

<br><b>Bugs Fixed</b>
&nbsp;&nbsp;- <i>workaround for memory leak on OS X with java 1.6.0_29</i>
&nbsp;&nbsp;- <i>views get sorted after adding new view</i>
